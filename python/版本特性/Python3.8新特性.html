# 一、新语法

# 1. 海象运算符

有一种新的语法（“海象运算符” :=），可以将值赋给变量作为表达式的一部分。例：

if (n := len(a)) > 10:
    print(f"List is too long ({n} elements, expected <= 10)")

# 2. f-strings

方便快速的进行调试，将在{}中打印表达式和表达式的值。

x = 3
print(f'{x*9 + 15=}')

输出

x*9 + 15=42

# 3. 限制位置参数

限制仅仅作用于分隔符“/”的左侧，"/"左侧的参数在函数调用时必须使用位置参数，即使定义时有参数是关键字参数（比如示例 1 和示例 3）。 注意：提供默认值的关键字参数一定位于最右侧。

示例1：

def pow(x, y, z=1, /):
    r = x ** y
    if z is not None:
        r %= z
    return r


if __name__ == '__main__':
    print(pow(2, 10))  # legal
    print(pow(2, 10, 2))  # legal
    print(pow(2, 10, z=2))  # illegal
    print(pow(2, y=10, z=2))  # illegal
    print(pow(x=2, y=10, z=2))  # illegal

示例2：

def hello(msg, x, y, /, z):
    print(f'{msg=}, {x=},{y=},{z=}')


if __name__ == '__main__':
    hello("1:", 1, 2, 3)  # legal
    hello("2:", 1, 2, z=36)  # legal
    hello("3:", x=1, y=2, z=3)  # illegal

示例3：

def hello(msg, x, y, z=None, /, m=None, n=None):
    print(f'{msg=}, {x=},{y=},{z=},{m=},{n=}')


if __name__ == '__main__':
    hello("1:", 1, 2, 3, 4, 5)  # legal
    hello("2:", 1, 2, 3, 4, n=5)  # legal
    hello("3:", 1, 2, 3, m=4, n=5)  # legal
    hello("4:", 1, 2, z=3, m=4, n=5)  # illegal
    hello("5:", 1, y=2, z=3, m=4, n=5)  # illegal
    hello("6:", x=1, y=2, z=3, m=4, n=5)  # illegal
