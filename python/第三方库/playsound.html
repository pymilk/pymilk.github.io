# playsound

请根据笔记的写作日期，选择性阅读。

作者： 放羊的大少爷
Home： joynoting.com
写作日期： 2018-05-13
系统： Mac
软件版本：
python@3.6.5
playsound@1.2.2
参考：
playsound

如果你有尝试找过python中播放音频的库，你会发现很多的库都很陈旧，或者仅支持最基本的wav、ogg等格式，一旦需要支持更多的格式，则需要额外的支持工具。

然而，经过我的不懈努力，终于找到一个不需要手动额外安装额外工具的库，那就是playsound。（其实它还是依赖于avbin）

# 一、安装

pip install playsound
1

# 二、示例代码

from time import sleep
from playsound import playsound


playsound('/path/to/a/sound/file/you/want/to/play.mp3')
sleep(200)  # <1>
1
2
3
4

<1> 注意：一定要加延时，否则程序一闪而过

# 三、Mac下的问题

上述的示例代码在Windows下完美运行，但是在Mac下则会出现以下错误：

from AppKit import NSSound ModuleNotFoundError: No module named 'AppKit'
1

当时天真的逐步溯源解决，发现一直解决不完，总有新的错误信息出现。

最终的完美解决办法：安装模块pyobjc

pip install pyobjc
1

pyobjc 项目旨在提供Python和Objective-C编程语言之间的桥梁。

# 四、其它的播放库

# 4.1 pygame

pygame中的music 模块可以播放音频，但是它对mp3的支持是有限的，部分mp3并不能播放，能播放的有些会变音，不过对wav和ogg是支持的。

推荐它的原因，主要是可以实现对音频的播放控制，比如音量调节、音频定位、播放暂停、循环播放等。

# 4.2 pyglet

pyglet中的media 模块可以播放音频，但是它只支持未压缩的wav格式。

如果需要支持更多的格式，需要安装 avbin。

同样也可以实现对音频的控制，比如音量调节、音频定位、播放暂停等。

# 五、小结

如果只是简单的播放音频，选用playsound，它支持大多数的音频格式。如果需要实现一些简单的控制，选用pyglet。

TIP

注意：以上的库在播放音频时，都需要加上延时或循环配合播放。
