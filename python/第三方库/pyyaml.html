# 二、读取

with open("sprites.yaml") as f:
    datas = yaml.load(f)
    
# 读取数据：中文
with open("sprites2.yaml", encoding="utf-8") as f:
    sprites = yaml.load(f)

需要注意的是5.1之后的版本已经弃用 load 方法，而改用以下的方法： You may also use one of the shortcut "sugar" methods:

  • yaml.safe_load
  • yaml.full_load
  • yaml.unsafe_load

所以以上代码的推荐写法是：

with open("sprites.yaml") as f:
    datas = yaml.safe_load(f)
    
# 读取数据：中文
with open("sprites2.yaml", encoding="utf-8") as f:
    sprites = yaml.safe_load(f)
