# 1. 参数为bool类型

parameters.toggle 中的参数会设置为 True 和 False，分别进行两次测试。

import slash

@slash.parameters.toggle('with_power_operator')
def test_power_of_two(with_power_operator):
    num = 2
    if with_power_operator:
        result = num ** 2
    else:
        result = num * num
    assert result == 4

# （2）多个参数-迭代形式

parameters.iterate 中的关键字参数的值为必须可迭代对象，依次取出参数 x，y 中的值进行测试，下面示例中，总共进行 9 次测试。

@slash.parameters.iterate(x=[1, 2, 3], y=[5, 6, 7])
def test_something2(x, y):
    assert x + y < 11

exclude() 原型如下：

# 三、测试前验证

slash.requires can receive either:

  1. A boolean value (useful for computing on import-time)
  2. A function returning a boolean value, to be called when loading tests
  3. A function returning a tuple of (boolean, message) - the message being the description of the unmet requirements when False is returned

简单的说，就是被要求验证通过的函数必须返回布尔值或者是一个包含布尔值的元组，这个元组必须只有两个元素。

# 被要求验证通过的函数的返回值
boolean
(boolean, message)
1
2
3

当被要求验证的函数不通过时，在为 False 时，将显示元组中的信息，然而我却没有显示。

def is_some_condition_met():
    ret = (False, "不通过")
    return ret


@slash.requires(is_some_condition_met)
def test_something():
    ...
