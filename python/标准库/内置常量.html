# 1. Ellipsis的写法

Ellipsis 对象是 Python 的內建常量。

Ellipsis 对象可以与 pass 类比：

Ellipsis 表示未定义的内容（空内容）
pass 表示空语句
1
2

Ellipsis 与 ... 字面相同：

>>> Ellipsis == ...
True
1
2

以下函数使用 pass 或 Ellipsis 对象时都可以达到相同的作用。

def hello():
    pass
1
2
def hello():
    Ellipsis

# 也可以写成下面形式    
def hello():
    ...

# 2. Ellipsis的作用

下面开始正式介绍 Ellipsis 的作用，它主要的作用有两个:

# 2.1 用来表示未定义的内容

上面说到，在定义空函数时，Ellipsis对象和pass语句都能起到相同的作用。

那么在初始定义一些不确定的变量时，我们除了使用None之外，又多了一种实现的方式，下面两条语句都能达到同样效果。

x = None
1
x = ...
1

# 2.2 用来表示省略的内容

以下只是其中一个实例，还有更多的表示省略的内容的场景。

>>> a = [1, 2, 3]
>>> a.append(a)
>>> a
[1, 2, 3, [...]]
