# MongoDB安装

请根据笔记的写作日期，选择性阅读。

作者： 放羊的大少爷
Home： joynoting.com
写作日期： 2018-12-21
系统： Mac@10.14.2
软件版本： MongoDB@4.0.4
Robo 3T@1.2.1
参考：
MongoDB
Robo 3T

数据库主要分为两种，关系型数据库和非关系型数据库。与关系型数据库相比，突出的优点和不足如下：

优点：

  • 非关系型数据的性能高于关系型，

不足：

  • 不支持事务和复杂的联合操作

爬虫比较适合使用非关系型数据库，而 MongoDB 就是非关系型数据库的代表，所以我选择了 MongoDB 作为爬虫的数据库。

本文主要是在 Mac 系统下使用 brew 方式安装。

# 一、安装

brew 方式安装：

>> brew install mongodb
1

配置文件默认在：/usr/local/etc/mongod.conf

通过 dbpath 参数设置：数据存放目录

# 二、启动

# 启动 mongodb
brew services start mongodb

# 进入 mongodb 控制台
sudo mongo

# 3. 创建用户

3.1 创建超级管理员

系统默认有三个内置数据库，其中 admin 为超级管理员数据库，这三个数据库都没有建立账户，我们可以先创建超级管理员用户。

使用以下命令可以查看所有的数据库。

>> show dbs
1

admin 数据库为超级管理员数据库，这个数据库的用户可以访问任何其他数据库的数据。

# 选择 admin 数据库
>> use admin

# 为 admin 数据库创建用户，用户名为 admin，密码为 admin，角色为 root
>> db.createUser({user:"admin", pwd:"admin", roles:["root"]})
1
2
3
4
5

创建完成之后，查看该数据库下的用户：

# 查看该数据库下的所有用户
>> show users
1
2

show users 只能看到该数据库的所有用户，不能检测密码，下面的命令可以验证用户名和密码是否匹配。

# 验证刚创建的 admin 用户，返回 1 表示验证通过，0 表示验证失败
>> db.auth("admin", "admin")
1
2

3.2 创建普通用户

# 选择 hello 数据库
>> use hello

# 为 hello 数据库创建用户，用户名为 hello，密码为 hello，角色为 dbOwner
>> db.createUser({user: "hello", pwd: "hello", roles: [{ role: "dbOwner", db: "blog" }]})
1

验证用户：

# 验证刚创建的 hello 用户，返回 1 表示验证通过，0 表示验证失败
>> db.auth("hello", "hello")
1
