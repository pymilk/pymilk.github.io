Home： joynoting.com

mongo Shell Methods

官方文档超级详尽，这里只做搬运，列举出常用的一些 shell 命令。

点击阅读 《MongoDB安装》

TIP

collection 替换为具体操作的 collection 名。

# 一、用户常用命令（User）

命令 意义
show users 显示当前数据库的所有用户
db.createUser() 为当前数据库添加用户
db.auth() 验证用户名和密码是否匹配
db.dropUser() 为当前数据库删除用户
db.updateUser() 为当前数据库更新用户数据

TIP

Deprecated since version 2.6: Use db.dropUser()instead of db.removeUser()

# 二、数据库常用命令（Database）

命令 意义
show dbs 显示当前 db 下的所有集合列表
use databaseName 指定 db
db.stats() 查看当前 db 的状态
db.createCollection() 在当前 db 下创建集合
db.dropDatabase() 删除当前 db
db.getName() 获取当前 db 名
db.getMongo() 获取当前 db 的链接地址

# 三、集合常用命令（Collection）

命令 意义
show collections 显示 collection 列表
use collectionName 指定 collection
db.collection.find() 显示 collection 中的所有数据
db.collection.count() 显示 collection 的记录数
db.collection.drop() 删除指定 collection

# 四、记录常用命令（Document）

命令 意义
增
db.collection.insert() 插入 N 条记录
db.collection.insertOne() 插入一条记录
db.collection.insertMany() 插入多条记录
删
db.collection.remove() 删除 N 条记录
db.collection.deleteOne() 删除一条记录
db.collection.deleteMany() 删除多条记录
改
db.collection.update() 更新 N 条记录
db.collection.updateOne() 更新一条记录
db.collection.updateMany() 更新多条记录
db.collection.save() 更新或插入一条(新)记录
查
db.collection.find() 查询 N 条记录
db.collection.findOne() 查询一条记录
db.collection.findOneAndDelete() 查询一条记录，然后删除
db.collection.findOneAndReplace() 查询一条记录，然后替换，未被替换的原字段数据会被清空
db.collection.findOneAndUpdate() 查询一条记录，然后更新
db.collection.replaceOne() 替换一条记录，未被替换的原字段数据会被清空
