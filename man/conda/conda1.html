# 一、Conda general commands

命令 意义
conda list 查看已安装的包
conda search 搜索指定包
conda install 安装指定包
conda uninstall 或 conda remove 卸载指定包
conda update 或 conda upgrade 更新指定包
conda clean 删除未使用的包和缓存
conda create 创建指定虚拟环境
source activate 激活指定虚拟环境（Windows 下为 activate）
source deactivate 使当前虚拟环境无效，激活默认虚拟环境 base （Windows 下为 deactivate）
conda info --envs 查看虚拟环境列表，效果同 conda env list，当前使用的环境有星号做标记

# 二、Managing packages

命令 意义
conda list 查看已安装的包
conda search 搜索指定包
conda install 安装指定包
conda uninstall 或 conda remove 卸载指定包
conda update 或 conda upgrade 更新指定包
conda clean 删除未使用的包和缓存

# 三、Managing environments

命令 意义
conda create 创建指定虚拟环境
source activate 激活指定虚拟环境
conda info --envs 或 conda env list 查看虚拟环境列表，当前使用的环境有星号做标记
conda remove 删除指定虚拟环境

# 四、Conda environment commands

使用 conda env 管理虚拟环境

命令 意义
conda env list 查看已安装的虚拟环境
conda env create 创建指定虚拟环境
source activate 激活指定虚拟环境（Windows 下为 activate）
source deactivate 使当前虚拟环境无效，激活默认虚拟环境 base（Windows 下为 deactivate）
conda env remove 删除指定虚拟环境
conda env update 根据环境文件更新当前环境
conda env export 导出指定虚拟环境

# 五、Example

# 5.1 Managing packages

  1. 从 Anaconda.org 安装指定包

    conda install -c pandas
    1

  2. 指定环境中安装包

    # 在 cling 环境中安装 pandas
conda install -n cling pandas
    1
    2

  3. 更新全部包

    conda update --all
    1

  4. 在 base 环境中更新 pandas 包

    conda update -n base pandas
    1

  5. 查看指定环境的包列表

    # 查看名为 cling 虚拟环境的包列表
conda list -n cling
    1
    2

# 5.2 Managing environments

  1. 创建指定虚拟环境

    # 创建名为 cling 的 Python 版本为 3 的虚拟环境
conda create -n cling
    1
    2

    如果不指定 Python 版本，默认版本与 base 一致

    # 创建名为 py36 的 Python 版本为 3.6 的虚拟环境
conda create -n py36 python=3.6
    1
    2

  2. 创建新环境并安装指定包

    conda create -n cling pandas
    1

  3. 克隆环境

    # 克隆一个现有的环境 cling，并命名为 myclone
conda create -n myclone --clone cling
    1
    2

  4. 创建一个完全一样的虚拟环境

    4.1 查看当前的包列表 （spec list）

    conda list --explicit
    1
     # 生成包列表文件
 conda list --explicit > spec-file.txt
    1
    2

    4.2 根据包列表文件创建新环境（spec list）

    # 根据 spec-file.txt 创建名为 cling 的新环境
conda create -n cling --file spec-file.txt
    1
    2

    4.3 根据包列表文件安装包（spec list）

    # 根据 spec-file.txt 为名为 cling 的环境安装包
conda install -n cling --file spec-file.txt
    1
    2

  5. 激活环境

    # Mac，激活名为 cling 的环境
source activate cling

# Windows，激活名为 cling 的环境
activate cling
