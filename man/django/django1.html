# django-admin和manage.py命令

请根据笔记的写作日期，选择性阅读。

作者： 放羊的大少爷
Home： joynoting.com
写作日期： 2020-03-14
系统： Mac@10.15.3
软件版本：
python@3.7.6
django@3.0.4
参考：
django-admin and manage.py

命令格式如下：

$ django-admin <command> [options]
$ python manage.py <command> [options]
$ python -m django <command> [options]
1
2
3

TIP

注意：如果对 settings 进行了拆分，则部分命令只能使用 python manage.py 方式。

序号 命令 意义
1 startproject 创建新的 Django 项目
2 startapp 创建新的项目 App
3 changepassword 修改内置用户模块 django.contrib.auth 用户密码
4 createsuperuser 为内置用户模块 django.contrib.auth 创建一个初始超级用户账户
5 makemigrations 从模型创建数据迁移文件，并保存在 App 的 migrations 文件件下
6 showmigrations 查看项目的所有迁移文件
7 migrate 根据最新的数据迁移文件，在数据库生成相应的数据表
8 runserver 在本机启动 Django 项目
9 collectstatic 搜集所有的静态文件
10 findstatic 查找静态文件的路径信息
11 check 检查项目是否存在异常
12 createcachetable 创建缓存数据表，为内置的缓存机制提供存储功能
13 dbshell 进入 Django 连接的数据库，shell 方式执行数据库的 SQL 语句
14 shell 进度 Django 的 shell 模式
15 diffsettings 显示当前 settings.py 与默认配置的差异，如果拆分了 settings.py 那这条命令将报错
16 dumpdata 以 json 文件格式导出数据表中的数据
17 loaddata 将数据文件导入数据库
18 flush 清空数据表中的数据信息
19 sqlsequencerest 重置数据表中的递增字段的索引值
20 remove_stale_contenttypes 删除数据库中不再使用的数据表
21 clearsessions 清空 session 数据
22 compilemessages 编辑语言文件，用于项目的区域语言设置
23 makemessages 创建语言文件，用于项目的区域语言设置
24 inspectdb 获取项目所有模型的定义过程
25 sqlflush 查看清空数据库的 SQL 语句脚本
26 sqlmigrate 根据迁移文件内容输出相应的 SQL 语句
27 squashmigrations 压缩迁移文件
28 test 运行 App 里的测试程序
29 testserver 新建数据库，并使用该数据库运行项目
30 sendtestemail 向指定的测试人发送测试的电子邮件
