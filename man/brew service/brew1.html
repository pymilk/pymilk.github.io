# brew service命令

请根据笔记的写作日期，选择性阅读。

作者： 放羊的大少爷
Home： joynoting.com
写作日期： 2018-03-08
系统： Mac@10.14.3
软件版本： brew@2.0.3-20
参考：
homebrew-services

Integrates Homebrew formulae with macOS' launchctl manager.

TIP

service_name 为指定服务名。

命令 意义
brew services start service_name 启动指定服务
brew services start --all 启动所有服务
brew services run service_name 运行指定服务
brew services stop service_name 停止指定服务
brew services restart service_name 重启指定服务
brew services list 显示已启动的服务
brew services cleanup Remove all unused services
