# 01. 环境变量

查看系统的 shell 解析器列表：

cat /etc/shells
1

查看当前使用的 shell 解析器

echo $SHELL
1

zsh 对应的文件 ~/.zshrc

bash 对应的文件 ~/.bash_profile

bash 最为常用，所以以下将针对 bash 进行介绍。

shell 脚本的组成：

  • if-else 和 for、while 等 shell 循环指令
  • grep、awk、cut 等文本处理工具
  • w、who、free 等二进制可执行程序

shell 脚本的使用目的：

  • 接受用户或文件传来的指令并在屏幕上显示数据
  • 可自动重复执行相同操作
  • 节约时间
  • 创建自己的 PowerTool 和 Power 实用工具
  • 管理员工作个性化
  • 减少设置服务环境和添加用户等操作中产生的错误
  • 等等

如需高级编程操作，请使用 gcc 或 Python。

bash 脚本关键字：

case fi then }
do for until [[
done function while ]]
elif if time

# 02. 基础

命令 意义
man 手册
echo 显示变量
; 按顺序执行命令，无论执行是否失败
&& 按顺序执行命令，遇到命令失败时终止
|| 按顺序执行命令，遇到命令成功时终止
! 配合首字母或字符串查询最近执行的命令
!! 重新执行最后的命令
!-n 重新执行倒数第 n 条命令，n 为具体数字

# 03. 目录操作

命令 意义
pwd 显示当前工作路径
ls 显示当前路径下的文件
less 浏览文本文件
more 浏览文本文件
cat 浏览文本文件
file 显示文件类型
cp 复制文件
mv 重命名或移动文件
rm 删除文件
mkdir 创建目录
rmdir 删除空目录
touch 创建文件
ln 创建链接

创建软链接

ln -s 源文件 目标文件
1

递归删除指定文件

find  . -name  '*.exe' -type  f -print -exec  rm -rf  {} \;
1

"." 表示从当前目录开始递归查找

“ -name '*.exe' " 根据名称来查找，要查找所有以.exe结尾的文件夹或者文件

" -type f " 查找的类型为文件

"-print" 输出查找的文件目录名

最主要的是是-exec了，-exec选项后边跟着一个所要执行的命令，表示将find出来的文件或目录执行该命令。

exec 选项后面跟随着所要执行的命令或脚本，然后是一对 {}，一个空格和一个 \，最后是一个分号。

find  . -name  '*.ad' -type  f -print -exec  rm -rf  {} \;
1

# 05. 文件访问权限

命令 意义
chmod 修改文件权限，+- rwx
su 临时转为超级用户/其他用户
chown 修改文件所有者
chgrp 修改文件的组所有者

如果出现 root 用户也无法删除的文件。先使用 lsattr 查看文件属性，再使用 chattr 删除文件属性，然后删除文件。

文件类型：

  • d：目录
  • -：文本文件、shell 脚本文件、可执行文件
  • c：字符设备
  • b：块设备
  • l：符号链接文件
  • s：socket 文件

# 08. 系统维护

命令 意义
users 显示当前登录的用户，等价于 who -q
who 显示当前登录的用户目录
groups 显示当前登录用户所属组信息
tset 初始化终端设置或显示终端类型
uname 显示系统内核信息
lsof 显示已打开的文件
arch 显示系统架构，等价于 uname -m
last 显示用户最后登录时间
strace 追踪命令执行时的系统调用和信号
ltrace 追踪命令执行时的库调用
nc 端口扫描
free 显示内存和缓存用量
lsdev 显示已安装的设备目录
du 显示磁盘文件使用情况
df 显示文件系统分区使用量
stat 显示文件各种详细信息
vmstat 显示虚拟内存的统计结果
netstat 显示当前网络统计和信息
uptime 显示当前系统运行的时间、系统不终止或重启的情况下继续运行的时间、当前登陆人数、平均负载等。若平均负载大于3，系统性能就会明显下降。
hostname 显示主机名
readelf 显示二进制文件信息
logger 记录系统日日志
pgrep 检索进程
top 显示系统整体状况
nohup 忽略所有 Hanghup 信号
service 开启和终止系统服务
ifconfig 显示和调试网络接口环境
route 显示或修改内核路由信息
tcpdump 实时显示网络数据包
mount 挂载文件系统
umount 卸载文件系统
env 显示环境变量或在其它系统环境中执行程序
watch 指定时间内重复执行该命令
nm 显示已编译而未清除的目标文件的符号目录
