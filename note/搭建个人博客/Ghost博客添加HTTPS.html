# Ghost博客添加HTTPS

请根据笔记的写作日期，选择性阅读。

作者： 放羊的大少爷
Home： joynoting.com
写作日期： 2018-11-19
系统： Mac
软件版本： Nginx@1.14.0
参考：

笔记主要讲解 Nginx 的 https 配置，所以购买 SSL 证书部分，由于过程比较简单，所以不会做详细讲解。 Nginx 的配置仅针对 Ghost 博客。

# 一、购买SSL

阿里云有免费SSL，可以使用一年。

  1. 登录到阿里云，产品与服务，安全，SSL证书（应用安全），购买证书。

  2. 选择购买如下配置的证书：

  1. 购买完成后，需要补全信息，补全自己的域名信息即可。需要注意的是，如果你的域名是在阿里云购买的，那么域名验证类型可以选择 DNS 验证，可以方便快速的通过域名验证。需要注意的是这种验证方式会在你的域名的 DNS 解析中多一条 TXT 记录。

  1. 如果域名验证成功，就可以选择下载证书使用了。

将下载得到的压缩包解压，会得到两个文件，文件后缀名分别为 .crt 和 .key。

证书文件名分别被我修改为 pymilk.com.crt 和 pymilk.com.key。

# 二、配置HTTPS

将下载得到的 SSL 证书上传到服务器，以我的为例，我将证书放置在 /home/https/ 目录下。

打开文件 /etc/nginx/sites-available/ghost.conf ，配置 Nginx 文件内容如下：

server {
        listen 80;
        server_name pymilk.com;
        # rewrite ^(.*) https://$server_name$1 permanent;
        return 301 https://$host$request_uri;
}

server {  
        listen 80;
        listen [::]:80 ssl ipv6only=on; 
        listen 443 ssl;
        listen [::]:443 ssl ipv6only=on;
        server_name pymilk.com;
        location / {
            proxy_set_header   X-Real-IP $remote_addr;
            proxy_set_header   Host      $http_host;
            proxy_pass         http://127.0.0.1:2368;
        }

        ssl on;
        ssl_certificate /home/https/pymilk.com.crt;
        ssl_certificate_key /home/https/pymilk.com.key;
}

至此配置完毕，重启 Nginx 即可。 上述配置可以实现 http 自动跳转 https 。

这里需要注意，正因为如此，所以 ghost 目录下的文件 config.development.json 中的参数 url 不能配置为 https 开头，一定要配置为 http，否则会导致重定向次数太多，无法访问自己的博客站点。

# 三、配置多个HTTPS

如果有多个域名，都想指向同一台主机，并且都想配置 SSL 证书，http 自动跳转 https 的话，可以参照我的配置。 我的证书都放置 /home/https/ 目录下。

以网站 vaporzfy.top、pymilk.com、pymilk.cn 为例，为 Ghost 博客做 Nginx 配置，文件内容如下：

#1、修改80自动提升为https
server {
        listen       80;
        server_name  localhost;
        return 301 https://$host$request_uri;
    }
#2、配置https的反向代理
# HTTPS server
server {
        listen       443;
        server_name  vaporzfy.top;       
        ssl                  on;
        #证书
        ssl_certificate      /home/https/vaporzfy.top.crt;
        ssl_certificate_key  /home/https/vaporzfy.top.key;
        ssl_session_timeout  5m;
        ssl_protocols  SSLv2 SSLv3 TLSv1;
        ssl_ciphers  HIGH:!aNULL:!MD5;
        ssl_prefer_server_ciphers   on;

        location / {
                proxy_set_header   X-Real-IP $remote_addr;
                proxy_set_header   Host      $http_host;
                proxy_pass         http://127.0.0.1:2368;
        }     
}

server {
        listen       443;
        server_name  pymilk.com;       
        ssl                  on;
        #证书
        ssl_certificate      /home/https/pymilk.com.crt;
        ssl_certificate_key  /home/https/pymilk.com.key;
        ssl_session_timeout  5m;
        ssl_protocols  SSLv2 SSLv3 TLSv1;
        ssl_ciphers  HIGH:!aNULL:!MD5;
        ssl_prefer_server_ciphers   on;

        location / {
                proxy_set_header   X-Real-IP $remote_addr;
                proxy_set_header   Host      $http_host;
                proxy_pass         http://127.0.0.1:2368;
        }     
}

server {
        listen       443;
        server_name  pymilk.cn;       
        ssl                  on;
        #证书
        ssl_certificate      /home/https/pymilk.cn.crt;
        ssl_certificate_key  /home/https/pymilk.cn.key;
        ssl_session_timeout  5m;
        ssl_protocols  SSLv2 SSLv3 TLSv1;
        ssl_ciphers  HIGH:!aNULL:!MD5;
        ssl_prefer_server_ciphers   on;

        location / {
                proxy_set_header   X-Real-IP $remote_addr;
                proxy_set_header   Host      $http_host;
                proxy_pass         http://127.0.0.1:2368;
        }     
}
