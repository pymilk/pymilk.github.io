# 使用hugo搭建个人博客

请根据笔记的写作日期，选择性阅读。

作者： 放羊的大少爷
Home： joynoting.com
写作日期： 2018-05-11
系统： Mac
软件版本：
hugo@0.40.2
参考：
hugo-quick-start

本文介绍hugo的简单使用，安装方式采用brew。

# 一、安装

brew install hugo
1

# 二、升级

brew upgrade hugo
1

# 三、查看版本

hugo version
1

# 四、创建新站点

hugo new site iblog
1

iblog为我命名的站点，可以自行命名

# 五、新增一个主题

创建站点后，默认是没有主题的，需要自行选择并下载主题。

依次执行以下命令：

cd iblog
git init
git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke
1
2
3

# 六、设置主题

echo 'theme = "ananke"' >> config.toml
1

也可以手动打开iblog目录下的config.toml文件，在文件末尾添加theme = "ananke"这一行。

# 八、预览

预览时预览草稿，即draft为true的文章：

hugo server -D
1

预览时不预览草稿，即draft为false的文章

hugo server
1

theme = "ananke"  # <4>

# 十、生成静态页面

  • 生成静态页面
hugo
1
  • 生成指定url主页的静态页面
hugo --baseUrl="http://vaporzfy.top/"
1
  • 生成指定主题的静态页面
hugo --theme=ananke --baseUrl="http://vaporzfy.top/"
1

如果在配置文件config.toml中已配置好参数baseUrl和theme，直接使用hugo生成静态页面，运行后，在iblog目录下会生成public文件夹，这就是你可以直接部署到github pages的源文件。

# 十一、部署

使用git部署，很简单，不赘述。
