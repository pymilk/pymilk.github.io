# 使用Ghost本地搭建个人博客

请根据笔记的写作日期，选择性阅读。

Home： joynoting.com
写作日期： 2018-11-19
系统： Mac
软件版本：
Nodejs@10.13.0
Ghost-CLI@1.9.8
Ghost@2.6.0
参考：
Supported Node versions
How to install Ghost locally
Ghost

前前后后使用过 typecho、wordpress、firekylin、hexo、hugo、ghost，时至今日，选定了 ghost，原因很简单——ghost发布博客的方式很简单，节省时间。

虽然 Ghost 不及 hugo 高效，不如 hexo 主题多，但是它发布博客简单，正是我需要的。 有兴趣可以查看我早期的这篇博文。

使用各框架搭建个人博客

# 一、安装Nodejs

安装 Nodejs 前，首先阅读官方文档，查看 Ghost 所支持的 Nodejs 版本。 点击查看 Supported Node versions

接下来使用 brew 安装 Nodejs，我选择的版本是v10.13.0。

>> brew install node@10
1

安装完成后查看自己的 Nodejs 版本

>> node -v
v10.13.0
1
2

# 二、安装Ghost

现在开始正式安装 Ghost，我采用的是全局安装 Ghost-CLI 的方式进行本地安装，这只是安装方式的一种，除此之外，你还可以选择 git 源码进行安装。

点击查看 官方源码

安装步骤如下：

  1. 全局安装 Ghost-CLI
>> npm install ghost-cli -g
1
  1. 本地安装Ghost

Ghost-CLI 安装完成后，cd 到一个空目录下安装 Ghost，该目录就是 Ghost 的安装目录。

注意： 以后执行 ghost start 命令需要 cd 到该目录下执行。

>> ghost install local
1

至此，Ghost 已安装完成，安装完成后，Ghost 已经默认启动，现在你就能够访问新站点。 前台：http://localhost:2368 后台：http://localhost:2368/ghost

登入后台设置管理员邮箱和密码后，就可以通过后台发布和设置自己的博客了。 当然你还可以通过安装 Ghost 的 App 进行后台管理，前提是你先设置好了后台的管理员邮箱和密码。

点击查看 Ghost Native Apps

# 三、Ghost常用命令

命令 意义
ghost start 启动Ghost
ghost stop 停止Ghost
ghost restart 重启Ghost
ghost log 查看日志
ghost ls 列出所有正在运行的Ghost博客
ghost update 升级Ghost

更多的 ghost 命令，请查看官方帮助文档。 点击查看 Ghost CLI

# 四、可能遇到的问题

如果采用源码安装的方式，在安装完成后，首次启动时，可能会遇到以下问题。

解决方法： 没有对数据库进行初始化，按照提示进行初始化即可。

>> npm install -g knex-migrator
>> knex-migrator init
1
2

注意： 还是在 Ghost 的源码目录下执行上面的命令。

最后说一句，更多的配置，比如配置指定的数据库（默认使用的数据库为 sqlite3）、自己制作主题、使用官方 API 等等，请阅读官方帮助文档。 点击查看 Ghost Documentation
