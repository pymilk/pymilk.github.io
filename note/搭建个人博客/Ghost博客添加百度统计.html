# Ghost博客添加百度统计

请根据笔记的写作日期，选择性阅读。

作者： 放羊的大少爷
Home： joynoting.com
写作日期： 2018-11-19
系统： Mac
软件版本：
Nodejs@Nodejs@10.13.0
Ghost-CLI@1.9.8
Ghost@2.6.0
参考：
百度统计帮助中心

Ghost 本身并不支持统计，所以只能自己添加脚本使站点支持统计。

# 一、添加

  1. 首先先到百度统计注册账号，使用百度搜索的账号是不可以的，这里需要另外注册。

  2. 注册后登陆到百度统计后台，两步添加自己的站点，很简单。

  1. 获取统计代码，准备手动安装到自己站点。

  1. 以 Ghost 默认的 casper 主题为例，进入你的 Ghost 安装目录，打开 content/themes/casper/default.hbs 文件，在页面底部的 </body> 标签之前添加百度统计的代码即可。
<script>
var _hmt = _hmt || [];
(function() {
  var hm = document.createElement("script");
  hm.src = "https://hm.baidu.com/hm.js?4a730a1dbeaefa7cb235a94fc1b70e5d";
  var s = document.getElementsByTagName("script")[0]; 
  s.parentNode.insertBefore(hm, s);
})();
</script>
