# Ghost博客添加归档

请根据笔记的写作日期，选择性阅读。

作者： 放羊的大少爷
Home： joynoting.com
写作日期： 2018-11-22
系统： Mac
软件版本：
Nodejs@10.13.0
Ghost-CLI@1.9.8
Ghost@2.6.0
参考：
Ghost

Ghost 本身并不支持归档，所以只能自己添加归档页使站点支持归档。

# 一、添加归档页

# 1. 新建文章

新建一篇文章，文章名为 Archives, 其实这个名字无所谓，随便起，文章内容为空，什么都不用写。

# 2. 设计归档页

以 Ghost 默认的 casper 主题为例，在 content/themes/casper 文件夹下新建一个文件名为 page-archives.hbs 的文件，将以下内容添加到文件中。（当然懂前端的同学可以在这个文件中为你的归档页进行设计）

{{!< default}}

{{#post}}

<header class="main-header">
    <nav class="container main-nav clearfix">
        <div class="main-nav__brand">
            <a class="blog-title" href="{{@blog.url}}">{{@blog.title}}</a>
        </div>
        {{#if @blog.navigation}}
            {{navigation}}
        {{/if}}
    </nav>
</header>

<main id='container' class='container' role='main'>
</main>

<script src="//cdn.bootcss.com/jquery/3.1.0/jquery.min.js"></script>
<script src="//cdn.bootcss.com/moment.js/2.14.1/moment.min.js"></script>
<script type = "text/javascript"> 


/**
 * 调用ghost API，完成文章归档功能
 * 所需组件：jQuery、moment.js
 * @ldsun.com
 */

jQuery(document).ready(function() {
	//获取所有文章数据，按照发表时间排列
	$.get(ghost.url.api('posts', {
		limit: 'all',
		order: "published_at desc"
	})).done(function(data) {
		var posts = data.posts;
		var count = posts.length;

        var html = "<div id='content' class='content col-sm-12 col-md-12 col-lg-12'><article class='archives'><section class='archives-content'></section></article><div>"
        $(html).appendTo('#container');

		for (var i = 0; i < count; i++) {
			//调用comentjs对时间戳进行操作
			//由于ghost默认是CST时区，所以日期会有出入，这里消除时区差
			var time = moment(posts[i].published_at).utcOffset("-08:00");
			var year = time.get('y');
			var month = time.get('M')+1;
			var date = time.get('D');
			if( date<10 ) date = "0"+date;
			var title = posts[i].title;
			var url = "{{@blog.url}}"+posts[i].url;
			//首篇文章与其余文章分步操作
			if (i > 0) {
				var pre_month = moment(posts[i - 1].published_at).utcOffset("-08:00").get('month')+1;
				//如果当前文章的发表月份与前篇文章发表月份相同，则在该月份ul下插入该文章
				if (month == pre_month) {
					var html = "<li><time>"+date+"日</time><a href='"+url+"'>"+title+"</a></li>";
					$(html).appendTo(".archives .list-"+year+"-"+month);
				}
				//当月份不同时，插入新的月份
				else{
					var html = "<div class='item'>"+"<h3><i class='fa fa-calendar fa-fw' aria-hidden='true'></i> "+year+"-"+month+"</h3><ul class='archives-list list-"+year+"-"+month+"'><li><time>"+date+"日</time><a href='"+url+"'>"+title+"</a></li></ul></div>";
					$(html).appendTo('.archives');
				}
			}else{
				var html = "<div class='item'><h2>"+"共计"+count+"篇文章"+"</h2><h3><i class='fa fa-calendar fa-fw' aria-hidden='true'></i> "+year+"-"+month+"</h3><ul class='archives-list list-"+year+"-"+month+"'><li><time>"+date+"日</time><a href='"+url+"'>"+title+"</a></li></ul></div>";
				$(html).appendTo('.archives');
			}
		}
	}).fail(function(err) {
		console.log(err);
	});
}); 
</script>

{{/post}}

# 二、设置样式

其实不设置样式也可以，它会使用系统默认的样式，默认的系统样式文件为 page.hbs。 如果想要自定义样式，可以在主题文件夹下自行设置。

以 Ghost 默认的 casper 主题为例，进入你的 Ghost 安装目录，打开 content/themes/casper/assets/css 文件夹，新建一个 css 文件，设置自己的样式。

# 1. 设置css

以我的为例，新建的 css 文件为 style.css，文件内容为：

.archives .item {
    padding: 5px 0 5px 5px;
    overflow: hidden;
    border-bottom: 1px solid #eee;
}
.archives h2 {
    font-size: 20px;
    font-weight: bold;
    margin: 0 0 5px 0;
    color: rgb(200, 72, 50);
}
.archives h3 {
    font-size: 17px;
    font-weight: bold;
    margin: 0 0 5px 0;
    color: rgb(46, 165, 201);
}
.archives-list {
    list-style: none;
    font-size: 14px;
    line-height: 20px;
    padding-left: 25px;
}
.archives-list li {
    padding: 3px 0;
    overflow: hidden;
}
.archives-list time {
    margin-right: 5px;
    color: #999;
    display: inline-block;
    width: 45px;
    font-family: monospace;
}
.archives-list a {
    text-decoration: none;
    color: rgb(35, 194, 141);
}
