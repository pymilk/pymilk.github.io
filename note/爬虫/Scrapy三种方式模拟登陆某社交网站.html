当使用 scrapy startproject 生成项目之后，需要对 setting.py 文件进行配置。 一定要配置 DEFAULT_REQUEST_HEADERS 参数伪装浏览器，还有 ROBOTSTXT_OBEY 参数设置为 False。

# Override the default request headers:

    "User-Agent": "Mozilla/5.0 (Macintosh; Intel Mac OS X 10_14_2) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/71.0.3578.98 Safari/537.36",
    "Accept": "text/html,application/xhtml+xml,application/xml;q=0.9,*/*;q=0.8",
}

# 一、POST登录

# 方法一：

默认情况下，爬虫使用 GET 请求 start_url 中的 URL。所以我们手动在 parse 方法中生成 POST 请求。 如果程序所在的目录下生成了 renren1-1.html 文件的话，说明成功登录。

# -*- coding: utf-8 -*-
import scrapy


class Renren1Spider(scrapy.Spider):
    name = "renren1-1"
    allowed_domains = ["renren.com"]
    start_urls = ("http://www.renren.com/PLogin.do",)

    def parse(self, response):
        data = {"email": "vaporzfy@sina.com", "password": "z3231450"}
        # FormRequest: Scrapy发送POST请求的方法
        for url in self.start_urls:
            yield scrapy.FormRequest(url, formdata=data, callback=self.parse_page)

    def parse_page(self, response):
        if response:
            with open("renren1-1.html", "w") as f:
                f.write(response.text)

# 方法二：

这种方法的关键在于重写 start_requests() 方法，发送 POST 请求。 如果程序所在的目录下生成了 renren1-2.html 文件的话，说明成功登录。

# -*- coding: utf-8 -*-
import scrapy


class Renren1Spider(scrapy.Spider):
    name = "renren1-2"
    allowed_domains = ["renren.com"]

    def start_requests(self):
        url = "http://www.renren.com/PLogin.do"
        data = {"email": "vaporzfy@sina.com", "password": "z3231450"}
        # FormRequest: Scrapy发送POST请求的方法
        request = scrapy.FormRequest(url=url, formdata=data, callback=self.parse_page)
        yield request

    def parse_page(self, response):
        if response:
            with open("renren1-2.html", "w") as f:
                f.write(response.text)

# 二、GET+POST登录

访问大型的网站，可能出了用户名和密码的字段外，还有一些额外的字段，这时候就需要先使用 GET 请求网站，然后从响应中提取必要的参数，最后再发送 POST 请求。 如果程序所在的目录下生成了 renren2.html 文件的话，说明成功登录。

假设表单中有额外的字段 pid，提取的代码如下。

# -*- coding: utf-8 -*-
import scrapy


class Renren2Spider(scrapy.Spider):
    name = "renren2"
    allowed_domains = ["renren.com"]
    start_urls = ("http://www.renren.com/PLogin.do",)

    def parse(self, response):
        # 提取登陆需要的参数，如果有参数
        # _pid = response.xpath("//pid").extract_first()

        data = {"email": "vaporzfy@sina.com", "password": "z3231450"}
        # FormRequest: Scrapy发送POST请求
        request = scrapy.FormRequest(response, formdata=data, callback=self.parse_page)
        yield request

    def parse_page(self, response):
        if response:
            with open("renren2.html", "w") as f:
                f.write(response.text)

# 四、提取cookie

赠送一个自己写的提取 cookie 函数。

def extract_cookie(cookie_str):
    # 提取 cookie 中的每一个键值对
    data = cookie_str.split(";")
    # 去除两边空格
    data = [i.strip() for i in data]

    # 生成将 cookie 改为字典格式
    cookie = {}
    for i in data:
        kv = i.split("=")
        cookie[kv[0]] = kv[1]

    return cookie
