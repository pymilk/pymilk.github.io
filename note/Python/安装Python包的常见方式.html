# 安装Python包的常见方式

请根据笔记的写作日期，选择性阅读。

作者： 放羊的大少爷
Home： joynoting.com
写作日期： 2019-06-12
系统： Mac@10.14.5
软件版本：
Python@3.7.2
参考：
Installing Packages

以下安装以 flask 为例进行安装讲解，按照个人推荐的优先级逐次进行介绍，推荐程度从上至下，依次递减。

# 一、pip安装

默认情况下 Python3 以上的版本已自带 pip，使用 pip 方式安装包是最为常见的方式。

  1. Installing from PyPI
pip install flask
1
  1. Installing from Github
pip install https://github.com/pallets/flask/archive/master.zip
或者
pip install -e git+https://github.com/pallets/flask.git
1
2
3

# 二、conda安装

conda 管理的库针对数据科学相关的包做了优化，性能上比 pypi 更好，比如 TensorFlow。

  1. 如果不是使用 anaconda 的话，需要安装 conda。
pip install conda
1
  1. 然后就可以使用 conda 管理库了
conda instal flask
1

# 三、源码安装

下载源码，然后切换都源码目录下。

python setup.py install
或者
pip install -e .
1
2
3

# 四、文件拷贝

  1. Installing from a python file

如果包是一个单独的 py 文件的话，直接把文件拷贝到 Python 所在目录的 lib 目录中。

$python_dir/lib
1
  1. Installing from tar.gz file (linux)
pip install flask.tar.gz
1
  1. Installing from whl file
pip install Flask-1.0.3-py2.py3-none-any.whl
1
  1. Installing from requirements.txt
pip install -r requirements.txt
1
